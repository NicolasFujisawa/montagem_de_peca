# montagem_de_peca
Aplicativo web que age como uma prova de conceito para treinamento prático em realidade virtual, através de uma plataforma mais acessível com um celular. O único input necessário é a movimentação da camera, algo possível com o google cardboard.

## tecnologias
Utilizamos das frameworks, A-Frame, Three-JS para a manipulação de elementos 3D no navegador, e browserify para gerar um arquivo estático com todas dependencias necessárias.
